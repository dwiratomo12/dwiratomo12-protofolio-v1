Tentang Saya

Nama saya adalah Dwi Setyo Wiratomo. Biasa dipanggil dengan nama dwi. saya seorang mahasiswa di Universitas Pembangunan Nasional Veteran Jakarta di jurusan Informatika semester 5. Saya sangat antusias dengan pemrograman web pada saat semester 4 telah berakhir. Saya mencoba belajar sendiri atau otodidak sebelum semester 5 dimulai. Karena pemrograman web adalah salah satu mata kuliah saya pada saat di semester 5 nanti. Saya mencoba belajar supaya ada bekal nantinya untuk semester 5. Saya belum pernah membuat website dan tidak tahu caranya. Saya sangat bersyukur mendapatkan kelas gratis belajar pemrograman web dasar dari dicoding. Ini adalah pertama kali saya membuat website walaupun masih belum menarik bagi saya

Hobi saya adalah berenang, sepak bola, dan bermain game. Biasanya, disaat saya suntuk dengan program atau tugas yang saya buat, saya bermain game untuk menghilangkan rasa stres. Kepribadian saya adalah pekerja keras, rajin, siap bekerja dalam tim, disiplin, dan bertanggung jawab.
